# Welcome to Blazor Spark!

Blazor Spark is a full stack micro framework built on top of Blazor Server. Our goal is to provide a fully featured web application framework to .NET developers, freeing you to create without worrying about the little things.

Blazor Spark has 3 goals
- Provide a better developer experience
- Provide out of the box tools to increase development speed
- Provide a consistent project structure
